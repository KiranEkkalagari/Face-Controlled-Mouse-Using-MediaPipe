# Face Controlled Mouse using MediaPipe

A real-time face-controlled mouse system developed using Python, OpenCV, MediaPipe Face Mesh, and PyAutoGUI.

## Features

- Smooth mouse cursor control using nose tracking
- Left double blink → Left click
- Right double blink → Right click
- Eye Aspect Ratio (EAR) based blink detection
- Real-time webcam processing
- On-screen blink and click counters

## Technologies Used

- Python
- OpenCV
- MediaPipe
- PyAutoGUI
